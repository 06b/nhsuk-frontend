Header

Find out more about the header component and when to use it in the NHS digital service manual.

Dependencies

For this component to work, you need to make sure you include the required JavaScript. Include the compiled JavaScript for all components nhsuk.min.js in your project.

Also, if using the autosuggestion functionality that the NHS.UK site uses then the typeahead JavaScript is required, which has a dependency on jQuery which must also be included.
